# analise-sentimentos-olist
Projeto de Ciência de Dados focado em Processamento de Linguagem Natural (NLP). O objetivo é classificar automaticamente o sentimento (positivo ou negativo) de avaliações de clientes do e-commerce Olist. Utilizei Pandas para manipulação de dados, NLTK para limpeza de texto e Scikit-Learn (Regressão Logística) para a modelagem preditiva.
