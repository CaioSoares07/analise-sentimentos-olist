# Análise de Sentimentos em E-commerce (Olist)

## Sobre o Projeto
Este projeto foi desenvolvido com o objetivo de automatizar a classificação de avaliações de clientes do e-commerce brasileiro Olist, utilizando técnicas de Processamento de Linguagem Natural (NLP) e Machine Learning.

## Tecnologias Utilizadas
* Python (Pandas, Scikit-Learn, NLTK)
* Google Colab
* Machine Learning: Regressão Logística

## Pipeline de Desenvolvimento
1. **Limpeza e Tratamento:** Preparação de dados brutos para remoção de ruídos.
2. **Engenharia de Recursos:** Transformação de texto em vetores numéricos com TF-IDF.
3. **Modelagem:** Treinamento de classificador para detectar polaridade (Positivo/Negativo).
4. **Validação:** Identificação e ajuste de vieses do modelo.



## Resultados
O modelo demonstra eficácia ao identificar corretamente o sentimento de frases contextualizadas.
